## Description
A web application that allows users to search for their favorite movies and view details about them.

## Features
- Search for movies by title.
- View movie details including synopsis, release date, and ratings.
- Responsive design for mobile and desktop.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/movie-search.git
   ```
2. Navigate to the project directory:
   ```bash
   cd movie-search
   ```
3. Open `index.html` in your browser.

## Usage
- Enter a movie title in the search box and hit enter to find your movie.

## API Key
To use the OMDB API, you need to replace the API key in the `js/index.js` file. Locate the following line:

```javascript
let key = "OMDB API"
```

Replace `OMDB API` with your own OMDB API key, which you can obtain by signing up at [OMDB API](http://www.omdbapi.com/apikey.aspx).
